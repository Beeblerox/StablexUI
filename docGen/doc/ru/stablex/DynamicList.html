class DynamicList<T> implements Dynamic<T>
List of any objects. Once requested, objects become accessible as properties of this list Objects constructor must take no necessary arguments. New object is created every time you request an undefined property. So dynamicListInstance.someProperty never returns null
public function new (cls:Class<T>) : Void
Constructor @param cls - class to use for new objects in list
public inline function get (key:String) : Null<T>
Get stored value, or create a new one
public inline function exists (key:String) : Bool
Check if given key exists
