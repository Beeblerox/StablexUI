class TweenSprite extends Sprite
class TweenSprite implements easy access to Actuate methods and manages registered eventListeners
override public function addEventListener (type:String, listener:Dynamic->Void, useCapture:Bool = false, priority:Int = 0, useWeakReference:Bool = false) : Void
Equal to Sprite.addEventListener except this ignores `useCapture` and does not support weak references.
public function addUniqueListener (type:String, listener:Dynamic->Void, useCapture:Bool = false, priority:Int = 0, useWeakReference:Bool = false) : Bool
Add event listener only if this listener is still not added to this object Ignores `useCapture` and `useWeakReference` @return whether listener was added
override public function removeEventListener (type:String, listener:Dynamic->Void, useCapture:Bool = false) : Void
Equal to Sprite.removeEventListener except this ignores `useCapture`
public function clearEvent (type:String) : Void
Removes all listeners registered for this event
public function hasListener(event:String, listener:Dynamic->Void) : Bool
Indicates whether this object has this listener registered for specified event type
public inline function tween (duration:Float, properties:Dynamic, easing:String = 'Linear.easeNone', overwrite:Bool = true, customActuator:Class<GenericActuator> = null) : IGenericActuator
Easy access to Actuate.tween for this object. Equals to Actuate.tween(this, ....). Parameter `easing` should be like this: 'Quad.easeInOut' or 'Back.easeIn' etc. By default it is 'Linear.easeNone'
public function tweenStop(properties:Dynamic = null, complete:Bool = false, sendEvent:Bool = false) : Void
Calls Actuate.stop() for this object. By default `complete` and `sendEvent` equal to false
public function free (recursive:Bool = true) : Void
Free object. Removes all registered eventListeners and children. Also removes itself from parent's display list. If `recursive` is true (by default), tries to call .free(true) for each child (TweenSprite instances only)
public function freeChildren(recursive:Bool = true) : Void
Removes children. If `recursive` = true (default) tries to call .free(true) for TweenSprite instances
